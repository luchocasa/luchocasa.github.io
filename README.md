## 🚀 Sobre este proyecto

Portfolio personal desarrollado con **HTML, CSS y JavaScript puro** — sin frameworks externos. Incluye animaciones CSS, scroll reveal, filtrado de proyectos y un formulario de contacto funcional vía `mailto`.

**🔗 Ver en vivo:** [luchocasa.github.io](https://luchocasa.github.io)

## ✨ Características del portfolio

- Diseño **dark mode** con paleta navy/cyan
- Animaciones de entrada con `IntersectionObserver`
- Barras de habilidades animadas al hacer scroll
- Filtrado de proyectos por categoría
- Sección de contacto con formulario funcional
- Totalmente **responsive** para mobile y desktop
